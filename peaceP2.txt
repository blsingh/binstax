     printed out later with lpr(1).[m
[m
     If the argument [4mfile[24m is given, [1mscript[0m saves all dialogue in [4mfile[24m.  If no file name is given, the typeâ€[m

     [1m-c,[0m [1m--command[0m [4mcommand[24m[m
             Run the [4mcommand[24m rather than an interactive shell.  This makes it easy for a script to capture[m
             the output of a program that behaves differently when its stdout is not a tty.[m
[m
     [1m-e,[0m [1m--return[0m[m

             The command will follow a symbolic link.[m

[m
     Certain interactive commands, such as vi(1), create garbage in the typescript file.  [1mScript[0m works best[m
     with commands that do not manipulate the screen, the results are meant to emulate a hardcopy terminal.[m
